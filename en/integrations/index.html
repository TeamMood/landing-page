Use TeamMood right in the tools you use every day

Get notifications by email, Teams, Slack or WebEx where it’s most convenient for you and your team. This way, it's low-effort to get started and keep using it each day.

Don't use another tool, it's straightforward

We all use a lot of different tools. Using a new one is always a difficult choice. That's why integrations are so essential to make the right choice.

You can get mood notifications by email, but TeamMood also integrates with the most popular messaging apps: Microsoft Teams, Slack, and Cisco WebEx.

Create your own integration with the API

If you have more specialized needs, we have an API to integrate TeamMood in your own tools or to fetch data to create custom reports.

Single-sign on (SSO) for enterprise customers

We provide SSO integration using the SAML2 protocol, either through Azure AD, Okta or even your custom SAML2 portal. It is part of our Enterprise plan.
