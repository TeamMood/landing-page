Utilisez TeamMood directement dans les outils que vous utilisez au quotidien

Recevez des notifications par e-mail, Teams, Slack ou WebEx là où c'est le plus pratique pour vous et votre équipe. De cette façon, il est facile de commencer et de continuer à l'utiliser chaque jour.

N'ajoutez pas un outil supplémentaire

Nous utilisons tous beaucoup d'outils différents. En utiliser un nouveau est toujours un choix difficile. C'est pourquoi les intégrations sont si essentielles pour faire le bon choix.

Vous pouvez recevoir les notifications pour partager votre humeur par e-mail, mais TeamMood s'intègre également aux applications de messagerie les plus populaires : Microsoft Teams, Slack et Cisco WebEx.

Créez votre propre intégration avec l'API

Si vous avez des besoins plus pointus, nous avons une API pour intégrer TeamMood dans vos propres outils ou pour récupérer des données afin de créer des rapports personnalisés.

Authentification unique (SSO) pour les entreprises clientes

Nous fournissons une intégration SSO à l'aide du protocole SAML2, soit via Azure AD, Okta ou même votre portail SAML2 personnalisé. Cela fait partie de notre plan d'entreprise.
